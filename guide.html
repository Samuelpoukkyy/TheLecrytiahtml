Comment Rejoindre l'Aventure Lecrytia

Suivez ces étapes simples pour installer et jouer à notre modpack épique !

Étape 1 : Prérequis

  • Assurez-vous d'avoir une version récente de **Java (JDK 17 ou supérieur)** installée sur votre ordinateur. Vous pouvez le télécharger sur le site officiel de Java.
  • Téléchargez et installez **CurseForge** (recommandé) ou le launcher **ATLauncher / MultiMC**. Ce sont les lanceurs les plus compatibles avec les modpacks.
  • Assurez-vous d'avoir Minecraft **Java Edition** (version 1.20.1 ou supérieure, ou la version spécifique de votre modpack si elle est différente) installée sur votre PC.

Étape 2 : Télécharger le Modpack Lecrytia

Nous recommandons l'utilisation de CurseForge pour une installation facile :

  1. Ouvrez votre lanceur **CurseForge**.
  2. Allez dans l'onglet "Parcourir les Modpacks".
  3. Recherchez "**Lecrytia**" dans la barre de recherche.
  4. Cliquez sur le modpack et appuyez sur le bouton "**Installer**".
  5. CurseForge téléchargera automatiquement tous les mods et configurera le profil Minecraft pour vous.

Si vous utilisez un autre lanceur, vous devrez peut-être télécharger le modpack manuellement depuis [lien de téléchargement du modpack, par ex. CurseForge.com] et l'importer.

Télécharger Lecrytia (Lien CurseForge/Autre)

Étape 3 : Lancer le Jeu

  1. Une fois l'installation terminée dans CurseForge (ou votre lanceur), cliquez sur "**Jouer**" sur le profil Lecrytia.
  2. Le lanceur Minecraft va s'ouvrir. Assurez-vous que le profil Lecrytia est sélectionné et cliquez sur "**Jouer**".
  3. Le premier lancement peut prendre un certain temps car le jeu doit charger tous les mods. Soyez patient !
  4. Félicitations ! Vous êtes prêt à explorer le monde de Lecrytia.

Conseils de Démarrage Rapide

  • **Allouez plus de RAM :** Lecrytia peut être gourmand. Nous recommandons d'allouer au moins **6-8 Go de RAM** à Minecraft via les paramètres de votre lanceur (CurseForge : Paramètres > Minecraft > Allocation de la mémoire).
  • **Rejoignez notre Discord :** Pour toute question ou aide, notre communauté est là !
